# FINAL PROJECT BAYUCARA GRUP F

## 📚 Deskripsi

Repositori yang berisi kode dan sumber daya untuk menyelesaikan misi pada Final Project Bayucaraka 2026.
Misinya adalah membuat **[mesin capit](https://id.wikipedia.org/wiki/Mesin_capit) otonomus** untuk memindahkan _payload_ ke _dropping zone_.
Project ini memanfaatkan ROS2 dan Micro ROS. Deteksi _paylaod_ dan _dropping zone_ menggunakan  OpenCV dan ArUco marker.

## 📁 Struktur Project

Project dibagi menjadi beberapa folder sesuai dengan kegunaan dan tempat eksekusinya.

| Folder | Konten |
| - | - |
| mcu | Kode yang dieksekusi pada micro controller unit |
| res | Foto dan video yang digunakan project |
| ros_ws | Workspace ROS2 yang dieksekusi pada host machine |
| script | kode pembantu yang dieksekusi di luar misi |
